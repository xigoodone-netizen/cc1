## 2. 算法逻辑深度解析

### 2.1 马尔可夫链 (Markov Chain)
马尔可夫链是本系统的“灵魂”算法之一。它不关注久远的过去，而是聚焦于“当前数字对下一期数字的直接影响力”。通过建立转移概率矩阵，系统能够识别出如“百位出现5后，下一期出现8的概率高达35%”这类隐性规律。

### 2.2 遗漏值回补 (Missing Value Compensation)
这是一种平衡算法。当一个数字连续多期未出现时，其在统计学上的回归压力会增大。我们的公式 `min(missing * 0.3, 8)` 确保了这种权重是平滑增长的，既不会因为一两期的遗漏就过度反应，也不会在极长遗漏时导致权重失控。

### 2.3 归一化与平滑 (Laplace Smoothing)
为了防止某个算法给出 0 概率导致整体模型失效，系统采用了改进的 **Laplace Smoothing**。通过动态平滑因子，系统能够确保即便在数据稀疏的情况下，预测结果依然保持严谨和稳健。

## 3. 综合预测流程：九层缩水

系统并非简单地将上述权重相加，而是通过以下流程进行“九层缩水”：

1.  **特征提取**：从原始采集流中提取9个维度的原始特征。
2.  **多维加权**：应用上述对比表中的权重进行初次评分。
3.  **位置拟合**：针对百、十、个位的独特性进行差异化修正。
4.  **趋势修正**：根据最新的3-5期走势进行动态权重加成。
5.  **归一化处理**：将原始分数转化为 0-100% 的概率分布。
6.  **逻辑校验**：剔除不符合基本统计逻辑（如全奇、全偶等极端分布）的低效组合。
7.  **精选过滤**：从1000种组合中缩水至高概率矩阵。
8.  **冲突检测**：确保百十个位的预测结果在整体组合上具有协调性。
9.  **最终输出**：输出您在界面上看到的红色高亮预测号码。

## 4. 结论与建议

*   **稳定性**：跟随关系和马尔可夫链提供了预测的“基石”，建议长期关注。
*   **爆发力**：遗漏值回补和趋势分析常能捕捉到意料之外的“冷门”或“热号”，适合博取高收益。
